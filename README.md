# PromptBench

> 提示词评估与优化工具 - 通过多模型并行测试、混合评估体系、自动迭代优化，找到最佳提示词版本。

## 特性

- **多模型并行评估**：一次评估测试多个模型对同一提示词的响应
- **混合评估体系**：规则评估（6分）+ AI评估（6分），全面评估文章质量
- **版本管理系统**：追踪所有历史版本和性能变化
- **自动优化建议**：基于评估结果生成下一版提示词
- **灵活迭代**：支持线性迭代和非线性分支

## 快速开始

```bash
# 1. 安装依赖
pip install openai python-dotenv

# 2. 配置环境
cp .env.example .env
# 编辑 .env 添加 API 配置

# 3. 运行评估
uv run evaluate_prompts.py --evaluate

# 4. 查看排名
uv run evaluate_prompts.py --ranking
```

## 核心命令

| 命令 | 说明 |
|------|------|
| `--evaluate` | 运行多模型评估 |
| `--evaluate --from-version N` | 基于指定版本评估 |
| `--ranking` | 显示历史版本排名 |
| `--create-version N` | 基于版本 N 创建新版本 |
| `--skip-optimize` | 跳过自动生成下一版 |

## 评分体系（10分制）

### 规则评估（5分）- 客观指标
- **字数**（1.5分）：是否在要求范围内
- **段落数**（1分）：段落数是否合理（5-20）
- **段落长度**（0.5分）：平均段落长度是否合理（30-150字）
- **结构**（1分）：中间是否有≥3个观点段落
- **小标题**（1分）：是否有小标题结构

### AI评估（5分）- 语义指标
- **开头质量**（1分）：是否直接入题，有吸引力，符合人设
- **经典引用**（1分）：引用是否自然恰当，与观点相关
- **内容深度**（1分）：是否有深度，有启发性
- **文笔流畅度**（1分）：是否流畅自然，有节奏感，有人味儿
- **情感共鸣**（1分）：是否能引发共鸣，打动人心
