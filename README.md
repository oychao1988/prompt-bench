# PromptBench

> 提示词评估与优化工具 - 通过多模型并行测试、质量+AI检测平衡评估体系、自动迭代优化，找到最佳提示词版本。

**特别优化**：针对中文内容设计，集成朱雀AI检测等中文检测工具。

## 特性

- **多模型并行评估**：一次评估测试多个模型对同一提示词的响应
- **平衡评估体系**：质量评估（6分）+ AI检测（4分），兼顾质量与人类化程度
- **中文AI检测**：支持朱雀检测、GPTZero等多种检测工具，专门优化中文检测
- **版本管理系统**：追踪所有历史版本和性能变化
- **自动优化建议**：基于评估结果生成下一版提示词
- **灵活迭代**：支持线性迭代和非线性分支

## 快速开始

```bash
# 1. 安装依赖
pip install openai python-dotenv requests

# 2. 配置环境
cp .env.example .env
# 编辑 .env 添加 API 配置

# 3. 运行评估（无API密钥也可运行，使用模拟检测）
uv run evaluate_prompts.py --evaluate

# 4. 查看排名
uv run evaluate_prompts.py --ranking
```

## 核心命令

| 命令 | 说明 |
|------|------|
| `--evaluate` | 运行多模型评估 |
| `--evaluate --from-version N` | 基于指定版本评估 |
| `--ranking` | 显示历史版本排名 |
| `--create-version N` | 基于版本 N 创建新版本 |
| `--skip-optimize` | 跳过自动生成下一版 |

## 评分体系（10分制）

### 质量评估（6分）

#### 规则评估（3分）- 客观指标
- **字数**（1.0分）：是否在要求范围内
- **段落数**（0.7分）：段落数是否合理（5-20）
- **段落长度**（0.3分）：平均段落长度是否合理（30-150字）
- **结构**（0.6分）：中间是否有≥3个观点段落
- **小标题**（0.4分）：是否有小标题结构

#### AI语义评估（3分）- 语义指标
- **开头质量**（0.6分）：是否直接入题，有吸引力，符合人设
- **经典引用**（0.6分）：引用是否自然恰当，与观点相关
- **内容深度**（0.6分）：是否有深度，有启发性
- **文笔流畅度**（0.6分）：是否流畅自然，有节奏感
- **情感共鸣**（0.6分）：是否能引发共鸣，打动人心

### AI检测（4分）- 人类化程度

**评分公式**：`检测分 = (1 - AI率) × 4`，保留2位小数

| AI率 | 人类率 | 检测分 | 评级 |
|------|--------|--------|------|
| 0% | 100% | 4.00 | ⭐⭐⭐⭐⭐ 完美 |
| 20% | 80% | 3.20 | ⭐⭐⭐⭐ 良好 |
| 40% | 60% | 2.40 | ⭐⭐⭐ 一般 |
| 60% | 40% | 1.60 | ⭐⭐ 较差 |
| 80% | 20% | 0.80 | ⭐ 很差 |
| 100% | 0% | 0.00 | ⭐ 纯AI |

**支持的检测器**（推荐按优先级配置）：
1. **朱雀检测**（腾讯）⭐⭐⭐⭐⭐ - 中文首选，特邀测试免费
2. **GPTZero** ⭐⭐⭐⭐ - 国际通用，中文支持良好
3. **CopyLeaks** ⭐⭐⭐ - 支持中文

详细配置见：[USAGE.md](USAGE.md) 或 [CLAUDE.md](CLAUDE.md)

## 中文AI检测配置

### 方案1：朱雀检测（推荐）

