# PromptBench

> 提示词评估与优化工具 - 通过多模型并行测试、规则引擎评分、自动迭代优化，找到最佳提示词版本。

## 特性

- **多模型并行评估**：一次评估测试多个模型对同一提示词的响应
- **规则引擎评分**：基于 8 维度自动评估生成质量，无需人工打分
- **版本管理系统**：追踪所有历史版本和性能变化
- **自动优化建议**：基于评估结果生成下一版提示词
- **灵活迭代**：支持线性迭代和非线性分支

## 快速开始

```bash
# 1. 安装依赖
pip install openai python-dotenv

# 2. 配置环境
cp .env.example .env
# 编辑 .env 添加 API 配置

# 3. 运行评估
uv run evaluate_prompts.py --evaluate

# 4. 查看排名
uv run evaluate_prompts.py --ranking
```

## 核心命令

| 命令 | 说明 |
|------|------|
| `--evaluate` | 运行多模型评估 |
| `--evaluate --from-version N` | 基于指定版本评估 |
| `--ranking` | 显示历史版本排名 |
| `--create-version N` | 基于版本 N 创建新版本 |
| `--skip-optimize` | 跳过自动生成下一版 |

## 评分体系（12分制）

- **开头**（2分）：是否直接入题
- **经典引用**（2分）：是否引用经典
- **结构**（1分）：中间是否有3个观点段落
- **结尾**（2分）：是否简短有力
- **字数**（2分）：是否在要求范围内
- **小标题**（1分）：是否有小标题结构
- **段落数**（1分）：段落数是否合理（5-20）
- **段落长度**（1分）：平均段落长度是否合理
