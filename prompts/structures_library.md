# 文章结构库

本文档定义了喜生公众号文章的所有可选结构。在生成文章时，根据选题和表达需要选择合适的结构。

---

## 结构一：场景感悟式（默认结构）

**适用场景**：人生感悟、处世智慧、独处安宁、读书笔记

**结构模板**：
```
开头：一个具体场景（3-5句）
- 直接入题，不拖沓
- 从一个具体画面切入（清晨河边散步、夜里等女儿电话、在老母亲床前守夜等）
- 明确点出"人过六十 / 老了才明白 / 老了才发现"等主题

中间：3-4个观点段落
- 每个段落从一个细节/故事出发，顺出一层道理
- 段落间空一行，方便阅读
- 每段控制在3-5句
- 分别围绕：
  * 和自己相处：身体、心态、节奏
  * 和家人相处：父母、伴侣、子女、外孙
  * 和世界相处：朋友、同事、老同学、这个时代

结尾：1-2句升华收束
- 简短有力，像写给同龄人的一小段慢慢的叮嘱
- 不开启新故事或论证
- 例："我们这个年纪的人，……也就够了"
```

**示例**：《半夜醒来，我才真正承认自己老了》

---

## 结构二：今昔对比式

**适用场景**：衰老反思、观念变化、时代记忆、教书余波

**结构模板**：
```
开头：一句话点出对比
- "以前...现在..."
- "年轻时...老了才..."
- 3-5句完成对比铺垫

中间：3个对比场景
- 每个场景都是"当年的我 vs 现在的我"
- 通过具体事件展现观念/心态的变化
- 每个场景200-300字

结尾：一句感悟
- 总结变化的核心
- 例："老了才发现，放不下的，从来不是儿女，是我们自己"
```

**示例**：《以前总觉得时间慢，现在觉得快》

---

## 结构三：对话展开式

**适用场景**：子女关系、老伴相处、朋友交往、问答互动

**结构模板**：
```
开头：一句对话/一个提问
- 直接引用对话原文
- 或以提问方式开头
- 3-5句引入

中间：对话展开 + 我的反应 + 事后的感悟
- 完整叙述对话场景
- 描述我当时的反应（说了什么/没说什么）
- 事后的反思和感悟
- 可以是单次对话，也可以是多次对话串联

结尾：一句劝慰
- 像对同龄人的轻轻叮嘱
- 不开启新话题
```

**示例**：《女儿说"挺好的"》

---

## 结构四：三段递进式

**适用场景**：深度思考、价值观反思、复杂议题

**结构模板**：
```
开头：一个现象/问题
- 提出要讨论的问题
- 描述一个现象
- 3-5句引入

中间：层层递进的3个认识
- 第一层认识（表面的理解）
- 第二层认识（深入的反思）
- 第三层认识（最终的释然）
- 每层150-200字，层层深入

结尾：一句总结
- 凝练三层认识的核心
- 例："我们这个年纪的人，能做的，不过是把家里的灯留着"
```

**示例**：《退休半年，我才慢慢懂了》

---

## 结构五：一事一议式

**适用场景**：具体事件、当天见闻、微感悟、短篇随笔

**结构模板**：
```
开头：一件具体的事（完整叙述）
- 时间、地点、人物、事件
- 完整叙述，200-300字
- 像讲故事一样，有细节有画面

中间：这件事引发的2-3个联想
- 由这件事联想到的其他场景
- 类似经历/他人故事/书中读到的内容
- 每个联想100-150字

结尾：一件事带出的一层道理
- 不宏大，但要真诚
- 例："老了才发现，……" "人到六十，……"
```

**示例**：《河边那对老夫妻》

---

## 结构六：问答体

**适用场景**：问答回信、读者互动、具体问题咨询

**结构模板**：
```
开头：一个问题
- 读者/老友问的具体问题
- 引用原话，或概括问题
- 3-5句引入

中间：我的回答（分3个小点）
- 第一点：我的类似经历
- 第二点：我当时是怎么处理的
- 第三一点：现在的反思和建议
- 每点150-200字

结尾：一句开放式的劝慰
- 不给绝对答案
- 留给读者思考空间
- 例："每家有每家的难处，怎么选，还要看你自己"
```

**示例**：《有读者问我"怎么和老伴相处"》

---

## 结构七：书信体

**适用场景**：写给特定对象、情感倾诉、跨时空对话

**结构模板**：
```
开头：称呼 + 寒暄
- "亲爱的XX："
- 简短寒暄，说明写信缘由
- 3-5句

中间：正文（分段叙述）
- 可以是回忆、倾诉、劝慰
- 结构自由，但保持克制温暖
- 每段不要太长，保持书信感

结尾：祝愿 + 署名暗示
- "祝你..." "愿..."
- 不真的署名，但保持书信语气
```

**示例**：《写给在外地的女儿》

---

## 结构八：日记体

**适用场景**：当天记录、碎片串联、日常感悟

**结构模板**：
```
开头：日期/天气（可选）
- "X月X日，晴"
- 或直接进入叙述

中间：一天中2-3个小场景
- 按时间顺序或自由串联
- 每个场景独立成段
- 场景间用过渡句连接

结尾：一天的感悟
- 轻轻收束
- 例："普通一天，也是福气"
```

**示例**：《今天的三件小事》

## 结构选择指南

### 按选题类型选择

| 选题类型 | 推荐结构 | 备选结构 |
|---------|---------|---------|
| 身体与衰老 | 场景感悟式 | 今昔对比式 |
| 伴侣与老伴 | 对话展开式 | 一事一议式 |
| 子女与牵挂 | 对话展开式 | 书信体 |
| 父母与送别 | 一事一议式 | 场景感悟式 |
| 朋友与社交 | 场景感悟式 | 今昔对比式 |
| 独处与安宁 | 场景感悟式 | 一事一议式 |
| 教书余波 | 今昔对比式 | 场景感悟式 |
| 时代记忆 | 今昔对比式 | 三段递进式 |
| 读书笔记 | 场景感悟式 | 三段递进式 |
| 读者问答 | 问答体 | 书信体 |

### 按字数需求选择

| 字数需求 | 推荐结构 |
|---------|---------|
| 300-500字 | 日记体、微感悟 |
| 500-800字 | 一事一议式、对话展开式 |
| 1000-1200字 | 场景感悟式、今昔对比式、三段递进式 |
| 1200字以上 | 场景感悟式（扩展至5段）、书信体 |

### 仿写爆文时

- 分析爆文的结构类型
- 从结构库中选择最接近的
- 或直接使用爆文的结构（不限于库内结构）

---

## 所有结构的共同原则

无论使用哪种结构，都必须遵守：

✅ **开头**：必须有具体画面，不空洞说理
✅ **中间**：每个段落都要从细节/故事出发，道理要"顺出来"
✅ **结尾**：克制而温暖，1-2句完成升华，不开启新话题
✅ **引用**：至少自然引用1-2句古诗词或经典名句
✅ **篇幅**：1000-1200字（微感悟/对话体/日记体可放宽至500-800字）
✅ **基调**：平和克制，不抱怨不煽情，带一点自嘲和温柔的幽默
✅ **段落**：段与段之间空一行，方便在公众号中阅读
