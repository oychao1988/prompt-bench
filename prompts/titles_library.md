# 标题库

本文档定义了喜生公众号文章的标题风格和模板。

---

## 标题核心原则

✅ **平实具体**：像一位老人慢慢说出来的话
✅ **有画面感**：能让人联想到具体场景
✅ **克制不煽情**：避免感叹号、夸张词
✅ **有代入感**：让读者觉得"这就是在写我"

---

## 标题模板（按类型分类）

### 一、感悟发现型

**模板**：《老了才发现/才明白/才知道，XXXX》

**示例**：
- 《老了才发现，身体不会等你》
- 《老了才明白，朋友越老越少》
- 《老了才知道，有些话该说的时候要说》
- 《老了才发现，真正心疼你的人，没几个》
- 《老了才明白，放不下的，是我们自己》

---

### 二、时间节点型

**模板**：《人过六十/退休后/到了这个岁数，XXXX》

**示例**：
- 《人过六十，学会体面地退场》
- 《人过六十，能把自己照顾好，就是福气》
- 《退休半年，我才慢慢懂了》
- 《退休后，我才真正开始活》
- 《到了这个岁数，不想勉强的事，就算了》

---

### 三、场景细节型

**模板**：《[具体场景]，我才[感悟/发现/明白]》

**示例**：
- 《半夜醒来，我才真正承认自己老了》
- 《女儿说"挺好的"，我知道她在硬撑》
- 《河边那对老夫妻，让我想起了什么》
- 《送走母亲那天，我才懂了》
- 《老伴炖的那碗汤，我喝了一辈子》
- 《外孙问我"爷爷你怎么不来我家"》

---

### 四、今昔对比型

**模板**：《以前[观念/做法]，现在[观念/做法]》

**示例**：
- 《以前总觉得时间慢，现在觉得快》
- 《以前爱热闹，现在怕孤单》
- 《以前总想改变别人，现在学会了随缘》
- 《以前觉得养生是老年人的事，现在发现》
- 《以前总觉得要有大房子，现在才明白》

---

### 五、对话引用型

**模板**：《[某人]说"[一句话]"，我[感受/感悟]》

**示例**：
- 《女儿说"你别来"，我知道她是忙》
- 《老伴说"你退休了别管我"，我才意识到》
- 《老母亲说"你别回来了"，我懂了》
- 《学生说"老师你变了"，我说"我们都变了"》
- 《外孙说"爷爷你老了"，我愣了一下》

---

### 六、一事一议型

**模板**：《[具体事件/物品]，XXXX》

**示例**：
- 《阳台上的那盆花》
- 《老伴住院那周》
- 《回老家的路上，我开始害怕》
- 《被拉进各种群之后，我慢慢退了出来》
- 《老同学聚会，我不想去了》
- 《重读〈傅雷家书〉》

---

### 七、温和劝慰型

**模板**：《我们这个年纪的人，XXXX》

**示例**：
- 《我们这个年纪的人，能做的，不过是把家里的灯留着》
- 《我们这个年纪的人，学会好好说再见》
- 《我们这个年纪的人，有些事，真的不能再等了》
- 《我们这个年纪的人，能把自己照顾好，就是福气》
- 《我们这个年纪的人，不想勉强的事，就算了》

---

### 八、读书感悟型

**模板**：《[书名/作者]，XXXX》

**示例**：
- 《书里的一句话，让我想起了女儿》
- 《重读〈傅雷家书〉，我才慢慢懂了》
- 《苏轼早就看透了》
- 《杨绛说"世界是自己的"，我想起了什么》
- 《〈论语〉里这句话，我教了一辈子，老了才真懂》
- 《汪曾祺说"人活着，就得有点兴致"》

---

### 九、疑问反思型

**模板**：《XXXX，我才知道/才明白/才发现》

**示例**：
- 《一个人吃饭，我才想起老伴的好》
- 《第一次坐公交有人让座，我才意识到》
- 《爬不动三楼的那天，我学会了一件事》
- 《视频挂了之后，我常常对着手机发呆》
- 《老友走了，我越来越怕参加葬礼》

---

### 十、简短意境型

**模板**：《[4-8字的意境/物品/场景]》

**示例**：
- 《慢慢变老》
- 《阳台上的花》
- 《一个人的散步》
- 《老伴的唠叨》
- 《老屋的灯》
- 《等女儿的电话》

---

## 标题生成公式

### 基础公式

```
[时间/人物/场景] + [才/才] + [发现/明白/知道] + [道理/感悟]
```

**示例**：
- 时间：老了/退休后/人过六十/到了这个岁数
- 人物：老伴/女儿/母亲/外孙/老友
- 场景：半夜醒来/河边散步/视频通话/菜市场

组合：
```
时间 + 道理：《人过六十，学会体面地退场》
场景 + 感悟：《半夜醒来，我才真正承认自己老了》
人物 + 对话：《女儿说"挺好的"，我知道她在硬撑》
书名 + 联想：《重读〈傅雷家书〉，我才慢慢懂了》
```

## 标题避坑指南

### ❌ 避免的标题风格

| 不要这样 | 不要这样 |
|---------|---------|
| 《震惊！老了才发现…》 | 夸张、震惊体 |
| 《人到六十必须做的5件事》 | 教条、清单体 |
| 《儿女不孝顺？因为你…》 | 指责、对立 |
| 《老伴走了，我好痛苦！！！》 | 卖惨、多感叹号 |
| 《退休老人的10个养生秘诀》 | 教程、干货风 |
| 《看完这篇，你会泪流满面》 | 煽情、诱导点击 |
| 《我是如何在退休后…》 | 成功学、自我炫耀 |
| 《现在不看，后悔一辈子》 | 紧迫感、营销号 |
| 《退休后一定要去的10个地方》 | 清单、攻略风 |
| 《这些话，我憋了一辈子》 | 爆款、情绪宣泄 |

---

### ✅ 推荐的标题风格

| 这样更好 | 这样更好 |
|---------|---------|
| 《老伴炖的那碗汤，我喝了一辈子》 | 具体物品，有画面 |
| 《女儿说"挺好的"，我知道她在硬撑》 | 对话开头，有代入感 |
| 《半夜醒来，我才真正承认自己老了》 | 场景+感悟，克制 |
| 《老了才发现，朋友越老越少》 | 平实，不夸张 |
| 《我们这个年纪的人，把家里的灯留着》 | 温暖，有共鸣 |
| 《重读〈傅雷家书〉，我才慢慢懂了》 | 读书+人生，有文化 |
| 《河边那对老夫妻》 | 简短，有想象空间 |
| 《老伴住院那周》 | 事件型，引发好奇 |
| 《以前总觉得时间慢，现在觉得快》 | 对比型，有共鸣 |
| 《慢慢变老》 | 极简，有意境 |

## 仿写爆文时的标题策略

### 爆文标题 → 喜生化改造

| 爆文标题 | 改造方向 | 喜生版本 |
|---------|---------|---------|
| 《老伴走了三个月，我好痛苦》 | 过度悲情→克制 | 《老伴住院那周，我才明白了一个道理》 |
| 《儿女不孝顺，养他们有什么用》 | 指责对立→理解 | 《女儿说"挺好的"，我知道她在硬撑》 |
| 《人老了，这三种人要远离》 | 教条对立→释然 | 《老了才发现，朋友越老越少》 |
| 《震惊！退休后才发现的真相》 | 夸张震惊→平实 | 《退休半年，我才慢慢懂了》 |
| 《看完泪流满面，父母的苦你知道吗》 | 煽情→克制 | 《八十岁的母亲叫我"小喜"，我突然鼻子一酸》 |

## 标题A/B测试建议

### 同一选题，准备2-3个标题

**选题**：老伴关系

| 标题版本 | 风格 | 测试目的 |
|---------|------|----------|
| 《和老伴吵了一辈子，老了才发现》 | 感悟发现型 | 测试"老了才"句式 |
| 《老伴炖的那碗汤，我喝了一辈子》 | 场景细节型 | 测试具体物品代入感 |
| 《老伴说"你退休了别管我"》 | 对话引用型 | 测试对话开头吸引力 |

**发布策略**：
- 第一篇用版本1，记录数据
- 如果阅读低于预期，下一篇用版本2
- 找到最适合该账号的标题风格

## 标题字数建议

| 平台 | 推荐字数 | 说明 |
|------|---------|------|
| 微信公众号 | 15-25字 | 完整表达，但有留白 |
| 今日头条 | 20-30字 | 可以稍长，信息量足 |
| 百家号 | 18-25字 | 平衡信息量和吸引力 |

**原则**：能短不短，能长不长，刚刚好。
